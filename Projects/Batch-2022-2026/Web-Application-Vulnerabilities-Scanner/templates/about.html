About VulnScanner

A simple scanner for identifying web application vulnerabilities

What is VulnScanner?

VulnScanner is a lightweight web application vulnerability scanner built with Python and Flask. It automates the detection of common security flaws in web applications, helping developers and security teams identify and fix vulnerabilities before they can be exploited.

Vulnerability Checks
  • Critical: SQL Injection
  • High: Cross-Site Scripting (XSS)
  • High: CSRF Vulnerability
  • High: Directory Listing
  • Medium: Missing Security Headers
  • Medium: Missing CSP
  • Medium: Clickjacking
  • Medium: Insecure Cookies
  • Low: Information Disclosure
  • Low: Server Version Exposed
  • Low: Mixed Content

  • Security: Werkzeug (password hashing)
