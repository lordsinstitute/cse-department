About CodeForge

What It Does

CodeForge is an Online Integrated Development Environment (IDE) that lets you write, run, and analyze code directly in your browser — no installations needed.

It features a professional-grade code editor (Monaco Editor — the same engine behind VS Code), real Python code execution, and an AI-powered code assistant.

How It Works
  1. Select a programming language from the dropdown
  2. Write or edit code in the Monaco Editor
  3. Click Run to execute Python code (output appears below)
  4. Click Ask AI to analyze, explain, refactor, or generate code
  5. Save code snippets for later use
  6. All sessions are logged in your history
AI Assistant (Dual Mode)
Mock AI Default

Works out of the box. Uses pattern-based code analysis — detects errors, explains code structure, suggests refactoring, and generates common code snippets. No API key needed.

Live AI Optional

Set GOOGLE_API_KEY in .env to enable Google Gemini AI. Provides real AI-powered code analysis, explanations, and generation with natural language understanding.
