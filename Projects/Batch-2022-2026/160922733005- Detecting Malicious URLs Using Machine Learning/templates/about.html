About URLShield

Project Overview

URLShield is a machine learning-based web application that detects malicious (phishing/harmful) URLs. Users enter a URL and the system extracts 28 numerical features from it, then uses a trained ML model to classify it as Legitimate or Malicious.

The project compares 8 different ML algorithms and uses the best-performing model for real-time predictions.
