About This Project

Project Overview

This project uses Machine Learning to predict whether a car insurance customer will file a claim based on their profile data.

The system analyzes 17 features including age, driving experience, vehicle type, credit score, and violation history to classify customers as Claim Likely or No Claim Expected.

This helps insurance companies in:

  • Risk assessment and premium calculation
  • Identifying high-risk customers
  • Fraud detection support
  • Policy underwriting decisions

How Prediction Works:
  1. User enters 17 customer features
  2. Categorical values are label-encoded
  3. Feature vector passed to trained ML model
  4. Model outputs probability of claim (0-1)
  5. Result displayed with confidence percentage
