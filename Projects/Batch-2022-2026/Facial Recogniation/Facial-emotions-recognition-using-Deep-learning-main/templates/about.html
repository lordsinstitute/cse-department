About the Project

Facial Emotion Recognition Using Deep Learning

Project Overview

This project implements a Facial Emotion Recognition System using a Convolutional Neural Network (CNN) built with PyTorch. The system can detect and classify 7 different human emotions from facial images in real-time.

The deep learning model analyzes facial features such as eyebrow position, eye shape, and mouth curvature to determine the expressed emotion. A face detection step using OpenCV's Haar Cascade ensures that only face regions are processed for accurate predictions.

7 Recognized Emotions
Emotion Visual Cues Description
Happy Raised eyebrows, wide eyes, upward mouth curve Expression of joy, pleasure, or contentment
Sad Lowered eyebrows, smaller eyes, downward mouth curve Expression of sorrow, grief, or unhappiness
Angry V-shaped eyebrows, narrow eyes, tight straight mouth Expression of displeasure, hostility, or frustration
Surprise High eyebrows, very wide circular eyes, O-shaped mouth Expression of shock or unexpected reaction
Fear Raised eyebrows, wide eyes, slightly open mouth Expression of anxiety, apprehension, or dread
Disgust Lowered uneven eyebrows, squinted eyes, wavy mouth Expression of revulsion or strong disapproval
Neutral Normal eyebrows, normal eyes, straight mouth Relaxed expression with no strong emotion
