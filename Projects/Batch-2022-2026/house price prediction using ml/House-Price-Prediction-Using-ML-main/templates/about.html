About This Project

Project Overview

House Price Prediction is a machine learning-powered web application that estimates California housing prices based on geographic, demographic, and economic features. The system trains multiple regression models and uses the best-performing one (Gradient Boosting) for real-time predictions.

How It Works

1. Input Features
Enter 9 housing characteristics (location, rooms, income, etc.)
2. Feature Processing
Categorical features encoded, numerical features normalized
3. Model Prediction
Gradient Boosting ensemble model predicts the price
4. Result
Predicted median house value displayed with input summary

ML Models Trained

  • Linear Regression — Baseline linear model
  • Ridge Regression — L2 regularized linear model
  • Decision Tree — Non-linear tree-based model
  • Random Forest — Ensemble of decision trees
  • Gradient Boosting — Sequential ensemble (Best Model)

Dataset

Based on the California Housing Dataset — a classic ML dataset containing information about housing in California census blocks.

  • 10,000 data points
  • 9 input features + 1 target
  • California geographic data
  • Median house values ($15K - $500K)

Security Features

  • Password hashing (PBKDF2-SHA256)
  • Session-based authentication
  • Parameterized SQL queries
  • Per-user prediction history
