📌 Spam Detection on Twitter using Deep Learning
📖 Overview

This project focuses on detecting spam tweets on Twitter using deep learning and natural language processing (NLP) techniques. It classifies tweets as spam or non-spam based on textual and user-based features.

🚀 Features
Data preprocessing and text normalization
Tokenization and sequence padding
Word embedding (GloVe, Word2Vec, Random)
Deep learning models (LSTM, CNN, Transformer)
Spam vs Non-Spam classification
📂 Dataset

The dataset includes:

User profile information
Tweet content
Followers and followings data
⚙️ Methodology
Data Collection
Text Preprocessing
Feature Extraction
Word Embedding
Model Training (LSTM-based)
Prediction & Evaluation
🧠 Model
LSTM Neural Network
Embedding Layer + Dense Layers
Sigmoid activation for binary classification
📊 Results
Achieved good accuracy in detecting spam tweets
Improved precision and specificity using deep learning models
🛠️ Technologies Used
Python
TensorFlow / Keras
NLP Techniques
Machine Learning

git clone <your-repo-link>

pip install -r requirements.txt

📌 Conclusion

This project demonstrates how deep learning can effectively identify spam content on social media platforms, improving online security and user experience.
