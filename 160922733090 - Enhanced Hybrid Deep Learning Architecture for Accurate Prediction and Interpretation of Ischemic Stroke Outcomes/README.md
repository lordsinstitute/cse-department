# 🧠 Brain Stroke Detection System using Deep Learning

## 📌 Project Title  
**Enhanced Hybrid Deep Learning Architecture for Accurate Prediction and Interpretation of Ischemic Stroke Outcomes**

---

## 📖 Overview  
This project presents a Deep Learning-based Brain Stroke Detection System that analyzes CT brain images to detect stroke conditions accurately.

The system uses a Convolutional Neural Network (CNN) model to assist medical professionals in making fast and reliable diagnoses, improving patient outcomes.

---

## 🎯 Objectives  
- Detect stroke from CT scan images with high accuracy  
- Assist doctors in early diagnosis  
- Reduce human error in medical image analysis  
- Provide a user-friendly web interface for predictions  

---

## 🧠 Key Features  
- Deep Learning-based image classification (CNN)  
- High accuracy (~97% validation accuracy)  
- Web-based interface using Flask  
- Real-time prediction system  
- Supports clinical decision-making  

---

## ⚙️ Technologies Used  
- Programming Language: Python  
- Framework: Flask  
- Libraries: TensorFlow, Keras, OpenCV, NumPy, Pandas  
- Frontend: HTML, CSS, JavaScript  

---

## 📊 Dataset  
- Total Images: 2,501 CT scans  
  - Normal: 1,551  
  - Stroke: 950  

---

## 🏗️ System Architecture  
1. Data Collection  
2. Image Preprocessing  
3. Feature Extraction  
4. Model Training (CNN / InceptionV3)  
5. Evaluation & Prediction  

---

## 📈 Model Performance  
- Training Accuracy: 98%  
- Validation Accuracy: 97%  
- High Precision, Recall & F1-score  
- Reliable ROC-AUC performance
